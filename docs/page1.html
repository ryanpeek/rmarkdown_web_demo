Interactive Plot

First let’s make a ggplot of cars, speed vs. distance.

# install.packages("plotly")
library(plotly)
library(ggplot2)

ggplot(data = cars, aes(x=speed, y=dist, color=speed)) +
  geom_line() +
  geom_point(alpha=0.5)

A simple ggplot of car speed vs. distance.

  • eval=FALSE is to show code, but not evaluate the code
  • echo=TRUE/FALSE is to show or not show the code
